
CdiCommons
=======

**Que es CdiCommons**

CdiCommons es un modulo que contiene clases abstractas


Instalacion
============

Installation via composer, agregar a composer.json:
```
"require" : {
    "cdi/cdicommons": "dev-master"
}
```



```
'modules' => array(
    'Application',
    'CdiCommons'             
)
```
